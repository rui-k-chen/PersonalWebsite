Data Visualization with Python Dash

In this project we have choosen the dataset from 2016 called "2016 Global Ecological Footprint." This dataset contianed 188 enteries, of which each 188 enteries represented a different country. The goal of this project is to use the Dash Python library to help us visualize data and make it interactive. We used libraries like pandas and dash for this project. We looked at factors like the human development index (HDI) and gross domestic product (GDP). The human development index according to the United Nations "Is a summary of average achievement in key dimensions of human development." It measures the life span, education levels, and standard of livings. The highest HDI that any country can achieve is 1.00. GDP shows "The monetary value of all finished goods and services made within a country during a specific period." If we graph the GDP and HDI on a scattergraph we will get this:



From the graph we can inference that there is a positive linear relationship between the countries' HDI and GDP. So typically if we see a higher GDP we can usually see a higher HDI as well. Specifically we can look at Luxembourg. Luxembourg's HDI is 0.89 which is fairly high. Consequently, Luxembourg also boasts the highest GDP on the list with 114.665l (in millions). Look at the other end of the spectrum we can also look at Niger. Niger has a lower GDP of 410.91 (millions) and has a HDI of only 0.34. In both cases we have support of the case that there is a positive linear relationship between HDI and GDP.

We then tie this into the ecological aspect of each country. Our common variable here is the countries. We look again to Luxembourg and see that while it has high HDI and GDP it is ecologically deficient. In fact it is the most ecologically deficient. A trend we tend to see is that the more far right (ecologically deficient) the more developed they tend to be. we can support the trend that the more developed country is the more ecological deficient it is. Looking at Burundi we see that althought it is not as developed it is more eco-friendly.



The next thing we did was look at the correlation matrix of each variable and see how they are related to each other. In order to visualize this we will use a heat map. First we did the correlation matrix in R using the cor() function. Of note we see that earth mass required has a very high correlation with carbon footprint of 0.95 with the highest value being 1. Another correlation we see is between earth required and ecological footprint of 0.99.



The final graph we did to that summerizes all the data is a parallel graph. Here each strand is a country and display throughout the graph is each individual variable.
